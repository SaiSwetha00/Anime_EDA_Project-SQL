# 📊 Anime Exploratory Data Analysis Using SQL

## 📌 Overview

Performed Exploratory Data Analysis (EDA) on an Anime dataset using SQL to analyze ratings, popularity, genres, episode counts, and anime categories.

## 🛠️ Tools Used

- SQL
- MySQL

## 🔑 Key Analysis

- Analyzed anime ratings and popularity trends.
- Identified top-rated and most popular anime.
- Compared average ratings across anime types.
- Explored genre distribution and episode statistics.
- Ranked top anime within each category using Window Functions.

## 💡 Skills Demonstrated

- Data Analysis
- SQL Querying
- Aggregate Functions
- GROUP BY
- Window Functions
- Ranking Analysis

## 🚀 Outcome

Generated meaningful insights from the anime dataset and strengthened practical SQL and data analysis skills.
